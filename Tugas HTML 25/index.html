Mengenal Semantic Element Di HTML 5

  • Article

Ketika ingin membuat layout untuk menampilkan blockartiker-artikel yang ada di website kita, tag artikel bisa kita gunakan dengan Element header dan footer.
Karena tag artikel memiliki type block,artinya elemen ini memiliki lebar sesuai browser jika kita ingin menentukan lebarnya.

  • Aside

Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman block maka kita bisa menambahlan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel.
Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
